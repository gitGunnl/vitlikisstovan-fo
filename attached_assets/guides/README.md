# Guide PDFs

This directory contains the PDF versions of the user guides.

## Required PDF Files

Please upload the following PDF files to this directory:

1. **ai-for-caretakers-guide.pdf** - AI for Caretakers Guide
   - This is the downloadable PDF version of the AI for Caretakers guide
   - Users will download this file when they click "Download PDF" on the caretakers guide page

2. **ai-for-kindergarten-guide.pdf** - AI for Kindergarten Guide  
   - This is the downloadable PDF version of the AI for Kindergarten guide
   - Users will download this file when they click "Download PDF" on the kindergarten guide page

## File Naming

The files must be named exactly as specified above for the download functionality to work correctly:
- `ai-for-caretakers-guide.pdf`
- `ai-for-kindergarten-guide.pdf`

## Upload Instructions

1. Create or export your PDF versions of the guides
2. Name them according to the specifications above
3. Upload them to this `attached_assets/guides/` directory
4. The download buttons on the guide pages will then work automatically
